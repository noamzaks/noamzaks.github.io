LyX Hebrew Guide

  • Install TeXLive using either install-tl-windows.exe for Windows or install-tl-unx.tar.gz for Linux/macOS. Note: I suggest you press Advanced and then change the scheme from full scheme (everything) to basic scheme (plain and latex), to download only the necessary LaTeX packages (you will be able to download additional needed ones later, and you may also choose to install the medium scheme).
  • Install the babel-hebrew, hebrew-fonts and culmus packages by running tlmgr install babel-hebrew hebrew-fonts culmus in an administrator command-line. Note: If you try to render a LyX document and you see an error like LaTeX Error: File `tikz.sty` not found. you will need to install an additional package. Either search Google for the relevant TeXLive package, or run the command tlmgr search --global --file tikz.sty and pick the package containing the exact file you need.
  • Install LyX by running the installer for your system which can be downloaded from here. For Windows, install the appropriate .exe file and for macOS install the .dmg file.
  • Update your global (non document-specific) preferences for hebrew by going into Tools->Preferences in Windows and Linux or LyX->Preferences on macOS.
    • Change Editing->Keyboard/Mouse to enable Use keyboard map and pick null for Primary and hebrew for Secondary.
    • Add a shortcut in Editing->Shortcuts by pressing New, enter language hebrew for the Function field and then enter your preferred shortcut in shortcut by clicking on it, pressing the keys and then click OK. I suggest Shift+Space for Windows/Linux and Control+Space for macOS.
    • In File Handling->File Formats pick PDF (pdflatex) for the default output format with TeX fonts.
  • Update your document preferences for hebrew (and defaults) by going into Document->Settings.
    • In Document Class pick Hebrew Article.
    • In Language pick Hebrew.
    • In LaTeX Preamble add the line \usepackage[use-david]{culmus}.
    • In Float Placement I suggest I suggest using Here definitely.
    • In Page Margins I suggest reducing the defaults to 1.5 for top and bottom and 1 for inner and outer.
    • Click Save as Document Defaults so all of the documents you create will use the above settings.
